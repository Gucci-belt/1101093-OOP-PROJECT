🎭 ฟีเจอร์ที่ 1: Mood-based Dining (กินตามอารมณ์)

═══════════════════════════════════════════════════════════════

【คอนเซปต์】
"วันนี้คุณรู้สึกยังไง?" ระบบจะแนะนำอาหารที่เหมาะกับอารมณ์ของคุณ
โดยใช้ Logic ง่ายๆ จากข้อมูลสารอาหารที่มีอยู่

【วิธีใช้งาน】
1. เปิดหน้า "หน้าหลัก" (Home)
2. จะเห็นส่วน "🎭 วันนี้คุณรู้สึกยังไง?" ด้านบน
3. กดปุ่มเลือกอารมณ์ที่ต้องการ:
   - 😰 เครียด
   - 😴 เหนื่อย
   - 💪 อยากผอม
   - 💧 บวมน้ำ
   - ✨ ทั้งหมด (แสดงทุกเมนู)

4. ระบบจะกรองและแสดงอาหารที่เหมาะสมทันที

【Logic การกรองอาหาร】

• เครียด (😰)
  → แนะนำของหวาน (Sugar ≥ 5g)
  → เรียงตาม Sugar จากมากไปน้อย
  → เหมาะสำหรับคนที่เครียดและต้องการความหวาน

• เหนื่อย (😴)
  → แนะนำอาหารที่มีพลังงานสูง (Kcal ≥ 400)
  → เรียงตาม Kcal จากมากไปน้อย
  → เหมาะสำหรับคนที่เหนื่อยและต้องการพลังงาน

• อยากผอม (💪)
  → แนะนำ Low Calorie (Kcal ≤ 350)
  → เรียงตาม Kcal จากน้อยไปมาก
  → เหมาะสำหรับคนที่กำลังควบคุมน้ำหนัก

• บวมน้ำ (💧)
  → แนะนำ Low Sodium (Sodium ≤ 600mg)
  → เรียงตาม Sodium จากน้อยไปมาก
  → เหมาะสำหรับคนที่ต้องการลดโซเดียม

• ทั้งหมด (✨)
  → แสดงรายการอาหารทั้งหมด
  → ไม่มีการกรอง

【UI/UX Features】
- ปุ่มมี hover effect (เปลี่ยนสีเมื่อชี้)
- ปุ่มที่เลือกจะมี active state (สีเขียว)
- การกรองทำงานแบบ real-time
- รายการอาหารอัปเดตทันทีเมื่อเลือกอารมณ์

⚔️ ฟีเจอร์ที่ 2: Food Battle (ศึกประชันแคลอรี่)

═══════════════════════════════════════════════════════════════

【คอนเซปต์】
เลือกอาหาร 2 อย่างมา "Battle" กันให้เห็นชัดๆ ว่าอันไหนอ้วนกว่ากัน
แสดงกราฟเปรียบเทียบแบบ Visual เพื่อให้เข้าใจง่าย

【วิธีใช้งาน】
1. เปิดหน้า "หน้าหลัก" (Home)
2. กดปุ่ม "⚔️ Food Battle - เปรียบเทียบอาหาร"
3. เลือกอาหารที่ 1 โดยกดที่กล่อง "อาหารที่ 1"
4. เลือกอาหารที่ 2 โดยกดที่กล่อง "อาหารที่ 2"
5. ระบบจะแสดงผลการเปรียบเทียบทันที

【ฟีเจอร์การเปรียบเทียบ】

• กราฟแท่งเปรียบเทียบ 3 หมวด:
  1. พลังงาน (Kcal)
  2. ไขมัน (Fat)
  3. น้ำตาล (Sugar)

• สรุปผล:
  - แสดงว่า "จานนี้อ้วนกว่า X เท่า!"
  - คำนวณจากพลังงาน (Kcal)
  - แสดงตัวเลขเปรียบเทียบชัดเจน

• Visual Features:
  - กราฟแท่งสีเขียวสำหรับค่าปกติ
  - กราฟแท่งสีแดงสำหรับค่าสูงสุด
  - แสดงตัวเลขบนกราฟ
  - Animation แบบ smooth เมื่อแสดงผล

【ตัวอย่างผลลัพธ์】
"ข้าวขาหมู" อ้วนกว่า "สลัดอกไก่" 2.2 เท่า! (690 vs 320 kcal)

🎰 ฟีเจอร์ที่ 3: Gacha Menu (สุ่มเมนูเสี่ยงดวง)

═══════════════════════════════════════════════════════════════

【คอนเซปต์】
"คิดไม่ออกกินไรดี?" กดปุ่มสุ่มเมนูแบบ Slot Machine
แก้ปัญหาโลกแตกของคนกินข้าวที่เลือกไม่ถูก

【วิธีใช้งาน】
1. เปิดหน้า "หน้าหลัก" (Home)
2. กดปุ่ม "🎰 สุ่มเมนูเสี่ยงดวง!"
3. รอ Animation หมุน (ประมาณ 1.5 วินาที)
4. ดูเมนูที่ได้ใน Modal
5. กดปุ่ม "เพิ่มลงตะกร้า" เพื่อเพิ่มเมนูที่ได้ทันที

【ฟีเจอร์พิเศษ】

• Animation:
  - ไอคอน 🎰 หมุนเมื่อกดปุ่ม
  - ปุ่มจะ disabled ชั่วคราวระหว่างสุ่ม
  - สุ่มจากรายการอาหารทั้งหมด

• Modal แสดงผล:
  - แสดงชื่อเมนูที่ได้
  - แสดงข้อมูลโภชนาการ (Kcal, Fat, Sugar, Sodium)
  - ปุ่ม "เพิ่มลงตะกร้า" เพื่อเพิ่มทันที
  - Design สวยงามด้วย gradient สีเหลือง

• การสุ่ม:
  - สุ่มจากรายการอาหารทั้งหมดในระบบ
  - แต่ละครั้งจะได้เมนูที่แตกต่างกัน
  - ไม่มีข้อจำกัดในการสุ่ม
